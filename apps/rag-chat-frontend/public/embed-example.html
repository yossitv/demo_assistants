Chat Widget Embed Example

How to Embed

Copy the iframe code below and paste it into your website where you want the chat widget to appear. Replace your-agent-id with your actual agent ID.

Embed Code:

<iframe
  src="http://localhost:3000/embed/your-agent-id"
  width="400"
  height="600"
  frameborder="0"
  allow="clipboard-write"
  style="border: none; border-radius: 8px;"
></iframe>
Note: Replace http://localhost:3000 with your production domain when deploying.

Responsive Embed

For a responsive embed that adapts to container width:

<div style="position: relative; padding-bottom: 150%; height: 0; max-width: 400px;">
  <iframe
    src="http://localhost:3000/embed/your-agent-id"
    style="position: absolute; top: 0; left: 0; width: 100%; height: 100%; border: none;"
    frameborder="0"
    allow="clipboard-write"
  ></iframe>
</div>
